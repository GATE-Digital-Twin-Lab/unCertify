# unCertify
This repository holds the codes developed under the project "Automatic maximally rigorous uncertainty quantification to enable design for certification (unCertify)". You can learn more about the project at uncertify.gate-ai.eu.

## Contents
There are no codes in this repo currently, but here's what to expect:

* A rigorous surrogate modelling suite
* An automatic uncertainty compiler
* An uncertainty contructor, powered by LLMs
